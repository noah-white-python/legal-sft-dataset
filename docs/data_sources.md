## 评估维度说明

| 维度 | 说明 |
|---|---|
| 数据量级 | 能拿多少?万级 / 十万级 |
| 反爬难度 | 无 / 轻(换 UA) / 中(登录+限频) / 重(验证码+加密) |
| 内容质量 | 问答结构清晰吗?有专业回答吗?噪声多吗 |
| 合规风险 | 允许爬吗?有隐私问题吗(法律数据要脱敏) |
| 获取成本 | 大概多久能拿到第一批 |
